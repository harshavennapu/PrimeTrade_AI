# PrimeTrade_AI
Key Findings

1. Sentiment affects trader profitability.
2. Extreme Greed/Fear periods show different trading behaviors.
3. Buy and Sell performance varies across sentiment regimes.
4. Some traders consistently outperform others.
5. Market sentiment can be used as an additional signal for trading decisions.

Conclusion

Trader performance is influenced by market sentiment. By combining sentiment indicators with trading data, traders can better understand market conditions and improve decision-making.
